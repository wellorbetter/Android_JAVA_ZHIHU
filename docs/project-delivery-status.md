## 与“完整知乎式项目”差距

### A. 后端能力差距
- 缺少 Auth 服务（JWT/RefreshToken）
- 缺少内容服务（提问、回答、评论、点赞、收藏）
- 缺少 Feed 服务（推荐流、关注流、热榜）
- 缺少分区/标签服务（Topic/Tag）

### B. 数据与爬虫差距
- 缺少采集任务调度器（定时、重试、限速）
- 缺少内容清洗、去重、审核与落库
- 缺少数据来源合法性控制（robots、版权、协议）

### C. 跨端工程差距
- 缺少统一 API 契约（OpenAPI）
- 缺少 Web/iOS 端
- 缺少统一鉴权与错误码体系
